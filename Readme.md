# 🏆 ContestHub

A dynamic contest management platform where users can participate in, create, and manage various competitions online.

## 🛠 Technologies Used
- **React.js** — Frontend framework
- **Node.js & Express.js** — Backend server
- **MongoDB** — Database
- **Tailwind CSS** — Styling
- **Firebase** — Authentication
- **Vercel** — Deployment

## ✨ Core Features
- 🏅 Browse and join contests
- ➕ Create and manage your own contests
- 🔐 Secure user authentication (Firebase)
- 📊 Leaderboard and result tracking
- 💳 Payment integration for premium contests
- 📱 Fully responsive design

## 🚀 Run Locally

**Step 1:** Clone the repository
```bash
git clone https://github.com/ruhi-akhi/ContestHubClient.git
```

**Step 2:** Go into the project folder
```bash
cd ContestHubClient
```

**Step 3:** Install dependencies
```bash
npm install
```

**Step 4:** Create `.env` file and add your environment variables
```env
VITE_FIREBASE_API_KEY=your_firebase_api_key
VITE_API_URL=your_backend_url
```

**Step 5:** Start the development server
```bash
npm run dev
```

**Step 6:** Open in browser
```
http://localhost:5173
```
